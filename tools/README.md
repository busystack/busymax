# Tools

Project maintenance scripts live here.

- `build_install_snap_local.sh` builds and installs a local Snap package. See
  the [Snap build and beta release guide](../docs/beta_snap_release.md) before
  using its scaffold workflow or publishing an artifact.
- `google_tasks_discovery/fetch_tasks_discovery.dart` refreshes the cached Google Tasks v1 discovery document and checks its locked revision.
